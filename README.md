# Initialiser `Prisma`

## Fichiers concernés

- `prisma/prisma.schema` ([voir le fichier](./e-commerce/prisma/schema.prisma))

## Solution

Pour initialiser `Prisma`, il suffit d'exécuter la commande :

```sh
npx prisma init
```

À la racine du projet Nest.
